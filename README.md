# Abhishek Kadam Portfolio Website

This is the portfolio website for Abhishek Kadam, a Senior Systems Engineer and Enablon Consultant. It is designed to be modern, responsive, and SEO-friendly.

## Features

*   **Responsive Design**: Looks great on all devices, from desktops to mobile phones.
*   **Modern Aesthetics**: Uses a clean dark mode with vibrant accent color (#e6155e) and glassmorphism effects.
*   **Performance Optimized**: Lightweight code, smooth scrolling, and optimized structure.
*   **SEO Ready**: Includes meta tags, JSON-LD structured data, sitemap, and robots.txt.
*   **Interactive UI**: Custom cursor, scroll reveal animations, sticky navbar, and hover effects.

## Technologies Used

*   **HTML5**: Semantic structure.
*   **CSS3**: Custom styling, animations, flexbox, and grid layouts.
*   **JavaScript**: Vanilla JS for interactivity (no heavy frameworks).
*   **Font Awesome**: Icons.
*   **Google Fonts**: 'Outfit' and 'Space Grotesk'.

## File Structure

*   `index.html`: Main portfolio content.
*   `style.css`: All styling rules.
*   `script.js`: Interactive JavaScript logic.
*   `robots.txt`: Search engine crawling instructions.
*   `sitemap.xml`: XML sitemap for search engines.

## How to Run

Simply open `index.html` in any modern web browser.

## Customization

To modify the content, edit `index.html`.
To change the color scheme, update the `:root` variables in `style.css`.
To add more interactivity or modify animations, edit `script.js` or `style.css`.
